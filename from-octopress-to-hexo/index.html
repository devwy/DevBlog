从 Octopress 迁移到 Hexo

2017-03-24

原来的博客是用 Octopress + GitHub Pages 搭建的，没找到好看的主题，一直用的默认主题。

在微博上看到 61.life 这个日记博客后，也想搭个博客记流水账，记得很早之前看到唐巧大神的一片博文 从 Octopress 迁移到 Hexo，因此看了下 Hexo 的文档和主题，决定就用 Hexo 搭建个博客，也把原来的博客也迁移过来。

博客的搭建和迁移还是很简单的，见 Hexo 的 文档。

原博客的文章被我删掉了。

主题

博客的主题用的是 Frost 的 PolarBear。喜欢这种极简的风格。

插件

  • PolarBear 主题需要安装插件 hexo-renderer-scss
  • 归档需要安装插件 hexo-generator-archive
  • RSS 需要安装插件 hexo-generator-feed。本站的 RSS 地址为 KiligWYu.com/atom.xml。
